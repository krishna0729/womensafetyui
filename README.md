🛡️ Women Safety UI Platform

A responsive and user-centric web application designed to promote women’s safety awareness, provide essential resources, and enable community engagement. This project was developed with AI-assisted tooling (Kiro AI) to accelerate development and ensure clean, maintainable code.

📌 Overview

The Women Safety UI Platform is a front-end focused application that delivers safety-related content through a structured and intuitive interface. It integrates informational pages, community interaction sections, and awareness blogs into a unified experience.

✨ Key Features
	•	Modular UI Architecture using reusable components
	•	Multi-page Navigation (Home, Blog, Community, Safety)
	•	Responsive Design for desktop and mobile devices
	•	Content-driven Pages for awareness and guidance
	•	Optimized Performance using Vite build tool

🛠️ Technology Stack

Category	Technology Used
Frontend	HTML5, CSS3, JavaScript
Framework	React (JSX)
Build Tool	Vite
Styling	Custom CSS
AI Assistance	Kiro AI

⚙️ Installation & Setup

To run this project locally:

# Clone the repository
git clone https://github.com/your-username/womensafetyui.git

# Navigate to project directory
cd womensafetyui

# Install dependencies
npm install

# Start development server
npm run dev

🌐 Deployment

The application is deployed using GitHub Pages for easy access and hosting.

Deployment Steps:

npm run build

	•	Upload the dist/ folder or configure GitHub Pages deployment.
